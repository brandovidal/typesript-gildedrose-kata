# Gilded Rose

This is the Gilded Rose kata in TypeScript.

## Getting started

Install dependencies

```sh
pnpm install
```

## Run the unit tests from the Command-Line

To run unit test from Vitest

```sh
pnpm run test
```

To run unit test from Vitest and generate report HTML

```sh
pnpm run test:ui
```
